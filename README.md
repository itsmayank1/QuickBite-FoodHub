# 🍔 QuickBite FoodHub

A full-stack food ordering web application built using Flask, MySQL, and Razorpay.

## 🚀 Features
- User Authentication
- Cart System
- Razorpay Payment Integration
- Wallet System
- Refund Management
- Admin Dashboard
- Delivery Agent Panel

## 🛠 Tech Stack
- Python (Flask)
- MySQL
- HTML, CSS, JavaScript
- Razorpay API

## 💡 Purpose
Built as a demo project during my Python Full Stack Internship to understand real-world web development.

## ⚙️ Setup
1. Clone repo
2. Install requirements
3. Setup database
4. Add Razorpay keys
5. Run app.py
