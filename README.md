# 🍔 QuickBite FoodHub

A full-stack food ordering web application built using Flask, MySQL, and Razorpay.

## 🚀 Features
- User Authentication (Login / Signup)
- Cart System
- Online Payment (Razorpay)
- Wallet System
- Refund Management
- Admin Dashboard
- Delivery Agent Panel
- Order Tracking System

## 🛠 Tech Stack
- Python (Flask)
- MySQL
- HTML, CSS, JavaScript
- Razorpay API

## 💡 Learning Purpose
Built as part of my Python Full Stack Internship to understand real-world web development.
